# My Dot files

## Requirements

### Install homebrew

```bash
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
```

### Install git

```bash
brew install git
```

### Install oh-my-zsh

```bash
curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh
```

### Install rvm

```bash
curl -sSL https://get.rvm.io | bash -s stable --ruby
```


## Installation

```bash
rake install
```
